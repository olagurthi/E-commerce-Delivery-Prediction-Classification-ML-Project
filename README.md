# 🚚 E-Commerce Delivery Prediction Classification

> 📊 A complete end-to-end **Machine Learning project** for predicting delivery outcomes in e-commerce logistics.

---

## 🚀 Overview

This project analyzes and predicts whether an order will be:

- ✅ **Delivered**
- ⏳ **Delayed**
- 🔄 **Returned**

Using a dataset of **50,000 orders**, we build a full ML pipeline including:

- 📈 Exploratory Data Analysis (EDA)
- ⚙️ Feature Engineering
- 🧹 Data Preprocessing
- 🤖 Model Training
- 📊 Evaluation & Visualization
